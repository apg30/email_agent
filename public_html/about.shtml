Technologies

No external files are used in YouWatt. All HTML, CSS and javascript components have been written by members of the group. A conscience effort has been taken to avoid using Bootstrap,jQuery or Online HTML templates.
HTML5: footer tag, header tag and nav tag.
CSS3: transitions, animations, background resize and flex box.
