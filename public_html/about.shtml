Overview

YouWatt is designed to be a simple email client with an inbuilt chat feature.

Bugs

We aim to make YouWatt as robust as possible. If you discover what you believe to be a bug then we strongly encourage you to report it as an issue on the github repository.


Contribution

This is an open source project. We encourage users to send pull requests if they are interested in contributing to the system. You can follow the github repository here.



Links

The website is comprised of the following pages:

Login/Register Allows the user to enter their credentials
Index The main interface from which the user can access their emails
Settings The settings page which allows the user to edit details about their account
About Meta information about the software.

Authors

This software was produced at Heriot Watt University by Aidan PJ Gallagher, Stuart J. Thain, Cameron A. Craig and Lee A. Hamilton.

Technologies

No external files are used in YouWatt. All HTML, CSS and javascript components have been written by members of the group. A conscience effort has been taken to avoid using Bootstrap,jQuery or Online HTML templates. The developers have tried to include new technologies from HTML5 and CSS3. Below are some the new components used in the system.
HTML5: footer tag, header tag and nav tag.
CSS3: transitions, animations, background resize and flex box.
