Welcome to the help page

This page is designed to help you use this web application, YouWatt. The below sections explain everything you need to know about YouWatt.

Sending an email

Click on the compose button on the top left of the webpage. Fill out the desired fields, click send.

Reading your emails

Navigate your way to the inbox, your email is displayed in a table. Click the update button to check for emails in the background (emails written to database). Refresh the page to reload email from the database.

POP3 Settings

Edit POP3 settings on the settings page, click Apply.

Delete personal data

Click 'Delete all personal data' on the settings page.

CSS/(X)HTML5 Validation

A stylesheet is made available at http://www2.macs.hw.ac.uk/~cac30/css/style.css. (X)HTML 5 Validation depends on the number of messages in the inbox. The page is sent in a POST request to the validation server, this is required as the deployment server is not publically accessible. If there are a lot of messages, the POST request is too large for the validator server to accept. To ensure success, the inbox can be emptied on the settings page.
